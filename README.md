## English Version
# Overview of the current state of Eudora Classic and solutions for critical outdated problems

This project offers information and tools for users of Eudora Classic (versions 3 to 7) who are experiencing problems due to the program's outdated nature.

The solutions focus primarily on:

* The lack of secure encryption protocols.

* The "Mojibake" issue caused by Eudora's incompatibility with UTF-8 encoding.

* Providing a comprehensive and detailed overview of the current state of Eudora and potential solutions for a modern, up-to-date version of Eudora Classic.


## What does this project include?

Two components ("Widgets") are provided:

* **Plugin (internal, DLL):** Adds functionality directly within Eudora.

* **EudoraToolbox (external, EXE):** An external tool that works with Eudora's TOC and MBX files.

* Note: Currently, these tools have a Spanish interface and are configured for very specific preferences, but they are functional for any Western language. They are offered as is and without warranty.
