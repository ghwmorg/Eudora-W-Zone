# Soluciones para problemas críticos de desactualización en Eudora Clásico

Este proyecto ofrece herramientas gratuitas para usuarios de Eudora Clásico (versiones 3 a 7) [cite: 16] que sufren problemas debido a la desactualización del programa. 

Estas herramientas solucionan principalmente:
* El "Mojibake" causado por la incompatibilidad de Eudora con la codificación UTF-8[cite: 18, 97, 100].
* Facilita la limpieza y formato de los mensajes[cite: 130].

## ¿Qué incluye este proyecto?
Se proporcionan dos componentes ("Widgets"):
* **Plugin interno (DLL):** Añade funcionalidades directamente en el menú de Eudora[cite: 125, 130]. Se instala colocando la DLL en la carpeta "plugins"[cite: 146].
* **EudoraToolbox (EXE):** Una herramienta externa para modificar archivos TOC y MBX con Eudora cerrado[cite: 125, 126, 131]. 

*Nota: Estas herramientas tienen la interfaz en español y están configuradas para mis preferencias personales, pero son funcionales para cualquier idioma occidental[cite: 132, 134, 135]. Se ofrecen tal cual, sin garantía, por lo que se recomienda encarecidamente trabajar en un entorno de pruebas o hacer copias de seguridad[cite: 137, 150].*
